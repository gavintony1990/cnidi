# MCP Server Catalog

本文件记录本项目推荐接入的 MCP 服务器类型、用途和风险等级。

## 推荐目录

| 服务器 | 用途 | 推荐范围 | 风险等级 | 备注 |
| --- | --- | --- | --- | --- |
| GitHub | issue、PR、commit、review | P0 | 中 | 优先作为需求入口 |
| Filesystem / local docs | 额外文档、共享规范、外部目录 | P0 | 中 | 注意目录权限范围 |
| Notion | PRD、会议纪要、规范 | P1 | 中 | 适合产品/需求协作 |
| Figma | 设计稿、Code Connect、设计系统 | P1 | 中 | 适合前端/UI 协作 |
| Sentry / Metrics | 故障定位、线上分析 | P1 | 高 | 建议只读 |
| Database | 数据查询、验证 | P2 | 高 | 默认只读，禁止直连生产写库 |
| Infra / Deploy | 发布、环境操作 | P2 | 极高 | 默认不接入项目共享配置 |

## 选择原则

- 优先选择能减少“人工复制上下文”的服务器。
- 优先选择只读或低风险服务器。
- 高风险服务器必须有强权限约束和人工确认机制。
- 如果某服务器只服务单人实验，不应放入项目共享 `.mcp.json`。
