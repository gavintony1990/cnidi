# Task to Implementation

## 目标

根据 `.claude/tasks/*.md` 拆解开发实现路径，并结合 rules / skills / agents 进入编码阶段。

## 步骤

1. 阅读任务文档
2. 阅读 `.claude/rules/README.md`
3. 选择对应规则：
   - 前端 shared + React / Vue
   - Java / Go / Python / C++
   - 数据库 shared + 具体数据库
4. 选择对应 skill
5. 选择对应 agent
6. 输出实现计划：
   - 后端改动
   - 前端改动
   - 契约同步点
   - 数据变更
   - 测试点

## 输出

- 实现计划
- 涉及模块
- 风险点
